# File Upload and Download Application

A simple and efficient web application that enables users to upload and download files securely. Built with Java and Spring Boot, this project demonstrates handling file storage and retrieval with ease.

## About

This project showcases my ability to implement file management features in a web application using Spring Boot. It covers file upload, download, and storage management while emphasizing security and usability.

## Features

- Upload files to the server with validation  
- Download files securely  
- List all uploaded files with metadata  
- Basic UI for file interactions  
- Error handling and user feedback

## Technology Stack

- Java  
- Spring Boot  
- Thymeleaf  
- Maven  
- Local file storage (can be extended to cloud storage)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/vivekchauhan2003/Upload-Download-Files.git
````

2. Configure and run the application:

```bash
./mvnw spring-boot:run
```

3. Access the application at:

```
http://localhost:8080/
```
